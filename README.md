# Trader Behavior vs Market Sentiment Analysis  
This project analyzes how trader profitability, trading volume, and behavior shift during market sentiment phases (Fear vs Greed).

├── ds_report.pdf
└── README.md

## How to Run
1. Open `notebook_1.ipynb` in Google Colab.
2. Run each cell in order.
3. Data is automatically loaded, cleaned, merged, and analyzed.

## Summary of Findings
- Profitability differs between fear and greed phases.
- Trading volume increases during greed.
- Win rates drop during fear.
- Buy/Sell imbalance shifts significantly during sentiment swings.

## Requirements
- Python 3
- Pandas
- Matplotlib
